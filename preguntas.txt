Paul: para que sirve el fetch?

lechuga: para actualizar las referencias del repositorio

Paul: con que comando se cambia de rama?

lechuga: git checkout nombre_de_rama

Paul: y entonces como creo una nueva rama?

lechuga: con el comando git Branch -b nombre_de_rama

Paul: y como subo todos mis cambios al repositorio?

lechuga: haciendo un "git add ." y un "git push"

Paul: ok, y si quisiera bajar un repositorio de GitHub con que comando debo hacerlo?

lechuga: el comando seria "git pull" o clonarlo con "git clone url_del_repositorio"
